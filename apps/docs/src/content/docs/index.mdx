## What is TurboPaste?

TurboPaste is a modern pastebin built on a TypeScript monorepo (Hono + tRPC + Prisma + Postgres + TanStack Router). It supports anonymous pastes, account-bound pastes that can be edited later, password protection, expirations, burn-after-read, and a clean REST API for automation.
